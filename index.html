vanilla Rough Notation

This is a small wrapper for the coolest of js libraries: Rough Notation

Animations are displayed automatically when the flag showOnLoad is set

They can also be launched programmatically by calling the show() method.

Rough notation group

By wrapping multiple rough notation elements into a group,

you can show them in animating in order

By default the library supports multi-line paragraphs. But if this is not wanted behavior, this can be disabled with `multiline="false"`. Here I am just adding padding to this line to make it span multiple lines. You can stop reading here.
